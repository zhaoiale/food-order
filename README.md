# 🍔 美食点餐系统
简洁美观、响应式的前端点餐网页，支持菜品选择、数量控制、订单计算、一键下单功能。

## ✨ 功能特色
- 菜品卡片展示（图片+名称+价格）
- 数量增减控制
- 订单实时计算总价
- 清空订单 / 提交订单
- 移动端 + 电脑端自适应
- 精美卡片动效、渐变主题

## 🚀 运行方式
1. 直接打开根目录 `index.html` 文件即可运行
2. 无需安装依赖，纯前端静态页面

## 📁 项目结构
food-order/
├── assets/ 静态资源
│ ├── css/ 样式文件
│ ├── js/ 脚本文件
│ └── images/ 菜品图片
├── index.html 页面入口
└── README.md 项目说明
