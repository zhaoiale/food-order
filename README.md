# 🍔 美食点餐系统
一款高颜值、功能完整、交互流畅的前端点餐网页，适配电脑/手机，真实还原线下点餐场景！

## ✨ 核心功能
- 🥢 菜品分类筛选（主食/饮品/小吃）
- 🌟 菜品详情查看（描述、评分、销量、库存）
- ❤️ 菜品收藏功能
- 🛒 购物车实时计算（数量/总价）
- 🗑️ 单个/清空订单操作
- 🏠 桌号选择 + 联系电话填写
- ✅ 确认下单弹窗 + 下单成功提示
- 📱 全端响应式适配
- 🎨 精美动画与视觉反馈

## 🚀 快速运行
1. 直接打开项目根目录 `index.html` 文件
2. 无需安装依赖，纯前端静态页面，开箱即用

plaintext

## 🛠 技术栈
HTML5 + CSS3 + JavaScript ES6+
响应式布局 | 模块化开发 | 交互优化

## 📝 使用说明
- 点击菜品卡片查看详情
- 点击 +/- 调整菜品数量
- 支持分类快速筛选菜品
- 填写桌号和电话即可下单
- 库存限制，防止超量点餐
